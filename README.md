# Peg-Moving-Game
### About
Peg Moving Game is a basic interactive game that moves a peg around a board. It is written entirely in Python.

### Execution
```Bash
python game.py [board_height] [board_width]
```
> Note: **board_height**

### Usage
#### Game Commands
* **up**
